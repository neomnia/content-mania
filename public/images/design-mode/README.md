## Où trouver ces images

### Option 1 : Logos officiels
- **Next.js** : https://nextjs.org/static/twitter-cards/home.jpg
- **Prisma** : https://www.prisma.io/images/logo.svg
- **Neon** : https://neon.tech/favicon/favicon.svg
- **Stripe** : https://stripe.com/img/v3/home/social.png
- **PayPal** : https://www.paypalobjects.com/webstatic/mktg/logo/pp_cc_mark_111x69.jpg

### Option 2 : Placeholder images
En attendant les vrais logos, vous pouvez utiliser des placeholders depuis :
- https://via.placeholder.com/120x60/CD7F32/FFFFFF?text=Next.js
- https://via.placeholder.com/120x60/CD7F32/FFFFFF?text=Prisma
- etc.

### Option 3 : Télécharger depuis les sites officiels
Visitez les sites officiels de chaque technologie et téléchargez leurs logos depuis leur page "Brand" ou "Press Kit".

## Alternative : Utiliser des icônes textuelles

Si vous préférez ne pas utiliser d'images, modifiez `app/(public)/page.tsx` pour afficher uniquement les noms des technologies avec des couleurs de marque.
